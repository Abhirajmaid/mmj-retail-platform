# Analytics Page UI Consistency

## What Changed
- Updated `apps/inventory/app/(back-office)/analytics/page.tsx` to match the UI patterns used across back-office pages like `stock`, `firm`, and `suppliers`.
- Added a top KPI section using `KpiCard`.
- Refactored chart/table/alert sections to use `CardHeader`, `CardTitle`, and `CardBody` from `@jewellery-retail/ui`.

## Why
- Keep spacing, card structure, and typography consistent across the dashboard for a smoother user experience.
